Firebase Setup Guide

Step-by-Step Setup

1

Create a Firebase Project

Go to console.firebase.google.com → Add Project → enter a project name → Continue.

2

Enable Firestore Database

In your project → Build → Firestore Database → Create database → Start in test mode → Choose a region closest to you (e.g. asia-south1) → Done.

3

Enable Email/Password Auth

Build → Authentication → Get Started → Sign-in method → Email/Password → Enable → Save.

4

Create Your Admin Account

Authentication → Users → Add user → Enter your admin email & password. This is how you log into this admin panel.

5

Get Your Firebase Config

Project Settings (⚙ icon) → Your Apps → Web app (</>) → Register app → Copy the firebaseConfig object.

6

Paste Config in Both Files

Open index.html and admin.html. Find the firebaseConfig object near the top of each file and replace the placeholder values with your actual config. Push to GitHub.

7

Set Firestore Security Rules

Firestore → Rules → Paste the rules from the project README. This allows public writes to leads (contact form) and authenticated reads/writes to reviews.

GitHub Pages Hosting: Push this folder to a GitHub repo → Settings → Pages → Deploy from branch (main / root). Your site will be live at https://YOUR-USERNAME.github.io/REPO-NAME/ and admin at .../admin.html.
